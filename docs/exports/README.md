# Exports Directory

This directory will contain:
- `model_results.json` - Model performance metrics
- `feature_importance.json` - Feature importance rankings
- `stats.json` - Dataset statistics
- `sample_wines.json` - Sample data for testing
- `confusion_matrix.npy` - Confusion matrix data
- `lgb_model.pkl` - Trained LightGBM model
- `scaler.pkl` - StandardScaler for feature normalization

These files are generated when you run the `wine-quality-ml-updated.ipynb` notebook.

**Note:** Model files (*.pkl) are gitignored to keep repository size small.
